Пет проект. Основная идея - облегчить жизнь людям с аллергией. Предполагается реализация двух интерфейсов, каждый из которых хранит независимый список аллергий. 
К первому в дальнейшем планируется прикрутить чат с нейросетью, чтобы можно было с учётом всех сразу всех или определённых категорий аллергий что-то спросить у нейросети
Второй интерфейс даёт возможность из представленного списка аллергенов выбрать желаемые. И с учётом тяжести аллергии, а так же удовольствия/необходимости данного алергена, будет методами оптимизации расчитано, 
что и в каком количестве лучше выбрать
